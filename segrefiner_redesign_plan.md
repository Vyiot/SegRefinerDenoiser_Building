# SegRefiner Redesign: Direct Denoising & Iterative Validation

This document describes the redesign of the SegRefiner model to follow a direct $x_t \rightarrow x_0$ mapping during training and an iterative $x_t \rightarrow \hat{x}_0 \rightarrow x_{t-1}$ sampling process during validation.

## 1. Training Logic: Direct $x_t \rightarrow x_0$ Mapping

The core goal is for the model to learn to recover the clean mask ($x_0$) directly from any noisy input $x_t$ in a single forward pass.

### Data Input: Pre-computed Noise Masks
- **Source**: Masks are loaded from `/home/ubuntu/vy/Denoiser/OEM_v2_Building/noise_mask/`.
- **Noise Levels**: Timesteps $t \in \{0, 1, 2, 3, 4, 5\}$ are sampled randomly (enabled by `use_x12=True` in the dataset config).
- **Format**: Each image has 12 versions (6 noise levels x 2 generation sources: `noise_gen` and `mix`).

### Training Target
- **Ground Truth**: Regardless of the input noise level $t$, the loss is computed against the Ground Truth binary mask ($x_0$).
- **Loss Function**: Binary Cross Entropy (BCE) + Texture Loss + Dice Loss between predicted $x_0$ and original $x_0$.

---

## 2. Validation Logic: Iterative Refinement

Validation uses an iterative process to gradually refine the mask from a coarse pseudo-label to the final output.

### Process: $x_t \rightarrow \hat{x}_0 \rightarrow x_{t-1} \rightarrow \dots \rightarrow x_0$
1. **Initialization**: Start with the initial pseudo-label (coarse mask). This is treated as $x_5$.
2. **Iteration Step**: For each step $i$ from 5 down to 0:
   - **Step A: Prediction ($x_i \rightarrow \hat{x}_0$)**
     The model takes the current image $I$ and noisy mask $x_i$ as input and predicts the estimated clean mask $\hat{x}_0$.
     $$\hat{x}_0 = \text{Model}(I, x_i, t=i)$$
   - **Step B: Re-noising/Interpolation ($x_i, \hat{x}_0 \rightarrow x_{i-1}$)**
     If $i > 0$, the next step's input $x_{i-1}$ is generated by mixing the predicted $\hat{x}_0$ with the original pseudo-label $x_{\text{pseudo}}$ using the noise level for $i-1$.
     $$x_{i-1} = \text{q\_sample}(\hat{x}_0, x_{\text{pseudo}}, t=i-1)$$
     *Note: Using the pseudo-label as the base ensures the refinement stays rooted in the initial estimate.*
3. **Final Output**: The result of the final step ($i=0$) is returned as the refined mask.
